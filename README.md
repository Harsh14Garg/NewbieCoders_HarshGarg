Project Description

Problem: In recent times, there has been an increase in the frequency of college fests and the events conducted as a part of the fests. Some of these events are fun, while others are educative like hackathons and workshops. But many students don’t get even informed about the events being conducted in their own colleges or in colleges nearby.

Solution: To solve the problem, we have created a platform which will allow the organisers of college level events to promote their event and get more recognition. The organisers of an event can post their event on our platform along with all the event details. The students/users can then view these events and book the tickets accordingly. The ticket booking details will be visible to the organisers.

Key Features: 
Feature 1: Single platform for both organisers to host events and the users to book tickets.
Feature 2: Users will receive the event tickets via a unique QR code which can be presented at the event venue to gain entry.
Feature 3: Users can filter events according to their location (city and college) and their preference (cultural, tech etc).
